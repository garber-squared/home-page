## Text Objects

| Shortcut   | Meaning                 |
|------------|-------------------------|
| `iw`       | inner word              |
| `it`       | inner tag (HTML)        |
| `i"`, `i'` | inner quotes, inclusive |
| `ip`       | inner paragraph         |
| `as`       | a sentence              |

## Tips

- Use the dot command.
- Use the more general text object, i.e. `iw` instead of `w`.
